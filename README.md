# 🚀 Ali Ahmed CC Suite (Pro Edition)

A professional-grade Python tool for generating and validating credit card formats using the **Luhn Algorithm**. Built for speed, accuracy, and ease of use in Termux.

## ✨ Key Features
- **Bulk Generation:** Generate 1000+ cards in a single hit.
- **Auto-Formatting:** Outputs in standard `CARD|MM|YY|CVV` format.
- **Luhn Validation:** Every card is mathematically verified before being saved.
- **Multi-BIN Support:** Add your own BINs for different countries/services.
- **Auto-Save:** All results are saved to `generated_cards.txt`.

## 🛠️ How to Install (Termux)
First, ensure you have Python installed:
```bash
pkg install python git -y
git clone https://github.com/frazking/CC-Gen.git
cd CC-Gen
```

## 🚀 How to Run
```bash
python ali_gen.py
```
